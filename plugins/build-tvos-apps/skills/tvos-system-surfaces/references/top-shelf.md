# Top Shelf

Top Shelf is an optional discovery surface for Apple TV home screen entry, not a substitute for the app's internal information architecture.

## Good fits

- featured content
- continue watching
- promotions with a clear deep link
- time-sensitive discovery surfaces that remain accurate

## Good defaults

- Keep the destination stable and immediate.
- Prefer content that still makes sense when the app opens some time later.
- Align Top Shelf items with the same content IDs the app uses internally.

## Guardrails

- Do not surface stale or low-confidence recommendations.
- Do not use Top Shelf as the only entry path into critical content.
